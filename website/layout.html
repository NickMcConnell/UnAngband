Welcome

Welcome to Unangband, my variant of Angband. Angband has a long history of development: I recommend you check out the excellent resources of www.thangorodrim.net and angband.oook.cz if you are interested in the history and current state of the play for this game.

I highly recommend you download the latest version of Unangband and start playing. Don't forget to upload your character dumps to the Angband Ladder at angband.oook.cz and get bragging...

Other Resources

  • The Berlios.de project page is at http://developer.berlios.de/projects/unangband/, which contains links to various downloads.
  • You can browse a web facing copy of the CVS at http://cvs.berlios.de/cgi-bin/viewcvs.cgi/unangband/.

Downloading from CVS

The easiest way to get a CVS copy is to download and install Eclipse. This will work for all platforms supported by the Eclipse environment (which runs within Java).
  1. Run Eclipse.
  2. Specify the workspace location you wish to keep the Unangband files (Accept the default if you are not sure).
  3. Choose Import from the File menu.
  4. Click on CVS, and choose Projects from CVS. Click Next.
  5. Create a new repository location with the details below. Click Next.
  6. Browse the existing modules and choose Unangband.
  7. Eclipse will then import the unangband module into the location you specified for your workspace.
The repository location details you'll need are:

Host: cvs.unangband.berlios.de
Repository path: /cvsroot/unangband
Authentication: anonymous
Password: none
If you have set up an account on berlios.de that has cvs access to Unangband, instead enter in the user name and password details of your account and change the Connection type to extssh.
